<Section
  title="Convention"
  description="Colors use a background / foreground naming convention. The background suffix is used for the background color and foreground for the text color."
>

```tsx
<div className="bg-primary text-primary-foreground">Hello</div>
```

All colors are defined as CSS custom properties using the OKLCh color space. This provides perceptually uniform color mixing and consistent contrast across palettes.
