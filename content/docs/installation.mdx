<Section
  title="Quick Start"
  description="The Formance design system ships as a shadcn registry plus a CLI. The CLI is the fastest way to add one, many, or all components."
>

```bash
pnpm dlx @formance/ds add button card input
pnpm dlx @formance/ds add --all
pnpm dlx @formance/ds list
```

Your project needs an existing shadcn setup (`pnpm dlx shadcn@latest init`).

</Section>

<Section
  title="@formance Namespace"
  description="If you prefer to install components via the shadcn CLI directly, add the Formance registry to your components.json:"
>

```json
{
  "registries": {
    "@formance": "https://ds.formance.com/r/{name}.json"
  }
}
```

Then install any component using the namespace:

```bash
npx shadcn@latest add @formance/button
npx shadcn@latest add @formance/card
npx shadcn@latest add @formance/input
```

</Section>

<Section
  title="Direct URL"
  description="You can also install components directly without configuring the namespace:"
>

```bash
npx shadcn add https://ds.formance.com/r/button.json
```

</Section>

## Prerequisites

- Tailwind CSS v4
- React 19+
- TypeScript 5+
- A `components.json` file configured for the default style
