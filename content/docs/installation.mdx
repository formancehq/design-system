<Section
  title="Quick Start"
  description="The Formance design system is distributed as a shadcn registry. Install components directly into your project using the CLI."
>

```bash
npx shadcn add https://design-system-three-mauve.vercel.app/r/button.json
```

</Section>

<Section
  title="@formance Namespace"
  description="To avoid repeating the full URL on each install, add the Formance registry to your components.json:"
>

```json
{
  "registries": {
    "@formance": "https://design-system-three-mauve.vercel.app/r/{name}.json"
  }
}
```

Then install any component using the namespace:

```bash
npx shadcn@latest add @formance/button
npx shadcn@latest add @formance/card
npx shadcn@latest add @formance/input
```

</Section>

<Section
  title="Direct URL"
  description="You can also install components directly without configuring the namespace:"
>

```bash
npx shadcn add https://design-system-three-mauve.vercel.app/r/button.json
```

</Section>

## Prerequisites

- Tailwind CSS v4
- React 19+
- TypeScript 5+
- A `components.json` file configured for the default style
